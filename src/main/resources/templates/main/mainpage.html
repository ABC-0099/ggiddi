우리의 해결책

게임처럼 몰입하는 한국어 학습

그래서 끼역띠귿은 RPG 게임, 실시간 퀴즈, 커뮤니티를 하나로 엮어 학습자가 스스로 계속하고 싶어지는 한국어 학습 경험을 만들었어요.

커리큘럼

단계별로 차근차근,
한국어를 완성해요

음절부터 일상 회화까지, 4단계 구성으로 기초부터 자연스럽게 쌓아갑니다.

🔤 1단계

음절

한글 자모음과 받침을 익히고 소리와 형태를 연결하는 기초 단계입니다.

📖 2단계

단어

일상에서 자주 쓰는 단어를 주제별로 학습하고 어휘력을 높입니다.

💬 3단계

문장

배운 단어로 문장을 만들고 한국어 문법 구조를 자연스럽게 익힙니다.

🗣️ 4단계

일상 회화

실제 생활 상황을 바탕으로 한 대화문으로 실전 표현력을 키웁니다.

학습 방식

배우는 게 재밌어야
오래 할 수 있어요

게임, 퀴즈, 커뮤니티가 하나로 연결된 몰입형 학습 경험을 제공합니다.

🎮

RPG 메타버스 게임

포켓몬 스타일의 2D RPG 게임으로 학습하며 캐릭터를 성장시켜요. 공부가 아니라 모험처럼 느껴집니다.

📝

단계별 퀴즈

각 단계를 마친 후 퀴즈로 실력을 확인해요. 틀린 문제는 다시 복습해 완벽하게 익힙니다.

👥

학습자 커뮤니티
